# IT5102E OpenGL project - Forest

![Screenshot](https://i.imgur.com/bf14MUA.jpg)

## Dependencies

* GLFW
* GLEW
* GLM
* SOIL
* Assimp

## Build instructions

```
> premake4 gmake
> make -j4  
> ./project.out
```
